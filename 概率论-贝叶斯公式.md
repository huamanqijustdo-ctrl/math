# 介绍

贝叶斯公式是概率论中的重要公式，它用于在已知新证据后更新事件的概率。其核心思想是根据先验概率和似然度计算后验概率，
公式通常表示为 P(A|B) = $\frac{P(B|A)P(A)}{P(B)}$

**公式解释** 

**P(A|B)**：后验概率在事件 B 发生的前提下，事件 A 发生的概率。

**P(B|A)**：似然度在事件 A 发生的前提下，事件 B 发生的概率。

**P(A)**：先验概率在考虑新证据（事件 B）之前，事件 A 发生的概率。

**P(B)**：证据的概率事件 B 发生的总概率。这可以通过全概率公式计算得出，即 P(B) = $\sum P(B|A_{i})P(A_{i})$


**公式推导** 

贝叶斯公式是由条件概率公式推导而来。 

条件概率公式：

P(A $\cap$ B) = P(B|A)P(A) 和 P(A $\cap$ B) = P(A|B)P(B)。

即两个事件同时发生的概率等于在已知其中一个事件发生的情况下，另一个事件发生的概率乘以那个事件的概率。

通过等式联立推导：由于两个公式都等于 P(A $\cap$ B)，因此可以令它们相等：P(B|A)P(A) = P(A|B)P(B)。将等式两边同时除以 P(B)，

即可得到贝叶斯公式：P(A|B)= $\frac{P(B|A)P(A)}{P(B)}$ 。
