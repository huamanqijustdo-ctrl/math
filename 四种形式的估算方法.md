# 拆分法

**适合格式**： A/B * c%

当A和B比较接近的时候（个人认为首先是位数相同，然后数值接近的情况），且A>B的时候，可以使用该方法。

例如：

```
65028/ 54112 × 15.7%

A 18.9%
B 17.2%
C 16.5%
D 15.4%
```

因为选项有效数位第二位不同,可以舍弃后两位变为：

```
650/541 * 15.7%

// 拆分
（541 + 109）/ 541 *  15.7%
// 大概是
（1 + 1/5）* 15.7% = 15.7% + 3.14% = 18.84
```

最终选A

# 灵活拆分法

**适合格式**： A/(1 + b%) * c%

也就是求增长量公式。

```
93652 / (1 + 17%) × 12.8%
   
A 11987
B 10246
C 9013
D 9638
```
求解过程如下：

```
93652     (117 + 11)%    93652
——————  * ———————————— = —————— * （1 + 11%/117）= 9365.2 + 936.42 = 10301.72
117%           10          10
```
