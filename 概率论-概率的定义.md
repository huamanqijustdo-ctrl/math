# 直观定义

概率的直观定义是随机事件发生可能性大小的数值表示。它可以用一个介于 \(0\) 和 \(1\) 之间的数来衡量，其中 \(0\) 表示事件不可能发生，\(1\) 表示事件必然发生。

# 古典定义

在随机试验的所有可能结果（样本空间）是有限且相互排斥，并且每种结果发生的可能性均相等的前提下，某一事件发生的概率等于该事件包含的基本事件的个数除以样本空间中基本事件的总个数。
这种定义常用于掷硬币、骰子等简单情景，也被称为古典概型或等可能概型。

**定义要点**

有限性: 随机试验的结果数量必须是有限的。

等可能性: 试验中的每一个基本结果发生的可能性必须是相同的。

# 统计定义

概率的统计定义：在同一组条件下所作的大量重复试验中，事件A 出现的频率总是在区间[0,1] 上的一个确定的常数p 附近摆动，并且稳定于p （频率的稳定值），则p 称为事件A 的概率，记作P(A) 。

# 公理化定义

概率的公理化定义，也称为柯尔莫果洛夫公理，是对概率进行数学严格定义的三个基本公理，用于描述概率空间 \(( $\Omega$ , $\Sigma$ ,P)\)。
这三个公理是：非负性（\(0 $\le$ P(A) $\le$ 1\)）、规范性（\(P( $\Omega$ )=1\)，其中 \( $\Omega$ \) 是样本空间）和完全可加性（对于互斥事件的集合，其并集的概率等于各事件概率之和）。 

概率公理化定义（柯尔莫果洛夫公理） 一个随机试验的样本空间为 \( $\Omega$ \)。若对任意事件 \(A\)，存在一个实数 \(P(A)\) 与之对应，则称 \(P(A)\) 为事件 \(A\) 的概率，且必须满足以下三条公理： 

**公理1：非负性**

 P(A) $\ge$ 0 
 
 概率必须大于等于 0，且小于等于 1 ( 0 $\le$ P(A) $\le$ 1)

**公理2：规范性**

P( $\Omega$ ) = 1

事件 $\Omega$ （即所有可能结果的集合）的概率为 1

**公理3：完全可加性**

对于任意一系列两两互斥的事件 $A_{1}$ , $A_{2}$ , $\dots$ ，
有 P( $A_{1}$ $\cup$ $A_{2}$ $\cup$ $\dots$ ) = P( $A_{1}$ ) + P( $A_{2}$ ) + $\dots$ 即，如果一系列事件互不相容，它们的并集发生的概率等于它们各自概率之和
