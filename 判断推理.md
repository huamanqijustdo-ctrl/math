# 1.混合多少

多多>少少
大大>小小

# 2.半真半假  

关键字：都只猜对一半

出现次数最少的主体为真

# 3.一句真话  关键字：只有一个为真

找矛盾句

# 4.两真两假 关键字：两人说话为真，两人说话为假

出现或/至少的句子为真；然后找矛盾句

# 5.两个有的

找到点名的那句（也就是有主语的句子），把主语改为所有，谓语反着说（如果原句说的是，就改为不是；原句说的不是，就改为是）

# 6.两个所有

找到点名的那句（也就是有主语的句子），把主语改为所有，谓语保持不变（如果原句说的是，就是；原句说的不是，就不是）

# 7.全对全错对一半 （当条件说有一人全对，一人全错，一人对一般的情况）

出现次数最多的主体为真

# 8.消消乐 （所有句子都是如果...那么，或者）
找无法配对的，（也就是单身狗）

# 9.三段论 （补充前提）

1) 两个有的->一个所有   即如果题干出现两个有些，则直接在选项找到有所有字眼的选项

2）去同存异    如果选项有多个选项有所有字眼，则在两个有的的句子里，把相同主体字眼去掉，留下的主体即为真

3）前在前，后再后   主体也主语和宾语，留下来的主体，要保持，原来是主语的还是作为主语，宾语还是宾语。

# 10.三推一   三个人中选一个

除了我其他都不是的为真


# 11 3+2 （题干出现5个人里，有3个怎么样，两个怎么样）

找到出现不同字眼的句子，找到重复出现的主体，即为真


# 12 4+3+2+1 （题干出现4个人里，3个人怎样，2个怎样，1个怎样）

找到相同话题的句子，找到重复出现的主体，即为真

# 13. 两难推理

A->B

-A->B

那么B一定为真


# 14.最大信息法

从出现最多话题的入手

# 15.选项提示法

所有选项中某一项都是相同的入手

# 16.组合排列比大小   简单来说就是出现需要排序或者比较大小的情况

从最大信息入手，再进行比较大小
