# 定义

条件概率是指在事件A已经发生的条件下，事件B发生的概率。 它通常表示为 (P( B|A ))，读作“在A条件下B的概率”。

其计算公式是 P(B|A) = $\frac{P(AB)}{P(A)}$ ，其中 P(AB) 是事件A和B同时发生的概率，而 P(A) 是事件A发生的概率。 

# 示例     

场景: 假设抛掷一枚六面骰子。 

事件A：骰子朝上的点数是偶数（{2, 4, 6}）。

事件B：骰子朝上的点数大于3（{4, 5, 6}）。 

问题: 在已知骰子朝上的点数是偶数（事件A）的条件下，点数大于3（事件B）的概率是多少？ 

解法: 

P(A) = $\frac{3}{6}$ = $\frac{1}{2}$ 

事件A和事件B同时发生是点数大于4且为偶数，即点数是4或6，因此 P(AB) = $\frac{2}{6}$ = $\frac{1}{3}$

根据公式 P(B|A) = $\frac{P(AB)}{P(A)}$ = $\frac{1/3}{1/2}$ = $\frac{2}{3}$  所以，在已知点数是偶数（2, 4, 6）的条件下，点数大于3的概率是 $2/3$
