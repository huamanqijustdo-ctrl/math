# 例题

小王参加“智力大冲浪”游戏,  他能答出甲、乙二类问题的概率分别为0.7和0.2, 两类问题都能答出的概率为0.1.  求小王两类问题都答不出的概率。

设事件 A 为“答出甲类问题”，事件 B 为“答出乙类问题”。

已知：
P(A) = 0.7, P(B) = 0.2, P(A $\cup$ B) = 0.1.

P(A)=0.7,P(B)=0.2,P(A $\cup$ B)=0.1.

需要求两类问题都答不出的概率，即 

P( $\overline{A}$ $\cap$ $\overline{B}$ ) = 1 - P( A $\cap$ B)。

计算 P( A $\cap$ B)：

P(A $\cup$ B)=P(A)+P(B)−P( $\overline{A}$ $\cap$ $\overline{B}$ ) = 0.7 + 0.2 − 0.1 = 0.8.

因此，

P( $\overline{A}$ $\cap$ $\overline{B}$ ) = 1− 0.8 = 0.2.

**疑问：为什么不能用A不发生的概率1-0.7，乘以B不发生的概率1-0.2求结果。**

这直接触及了概率计算中一个常见的误区：事件的独立性。

用 (1−0.7) × (1−0.2) = 0.3 × 0.8 = 0.24 ——这仅在事件“答不出甲类问题”和“答不出乙类问题”相互独立时才成立。

**如何判断是否独立？**

两个事件 A和B 独立的定义是：

P( A $\cap$ B ) = P(A) ⋅ P(B)

如果上式成立，则独立；

否则不独立。

代入本题数据：

P(A)⋅P(B) = 0.7×0.2 = 0.14

但题目给出的 P( A $\cap$ B ) = 0.1，两者不相等。

结论：事件 
A和 B 不独立，因此它们的补事件 $\overline{A}$ 和 $\overline{B}$ 一般也不独立，所以不能简单相乘。
